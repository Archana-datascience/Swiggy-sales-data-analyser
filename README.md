# 🍽️ Swiggy Sales Analysis Dashboard (Excel)

## 📌 Project Overview

This project presents an interactive **Swiggy Sales Analysis Dashboard** built in **Microsoft Excel** using **Power Query, Power Pivot, Pivot Tables, Pivot Charts, and Slicers**. The dashboard provides insights into sales performance, customer ratings, order trends, regional sales, and restaurant performance, enabling data-driven business decisions.

---

## 🎯 Objective

The objective of this project is to analyze Swiggy sales data and identify key business insights such as:

* Overall sales performance
* Order trends across months and weekdays
* Regional sales distribution
* Food category contribution
* Top-performing restaurants
* Top-performing cities
* Customer rating analysis

## 📌 Key Business Insights

### Sales Performance

* Total revenue generated was **₹5.30 Crore** from **197,430 orders**.
* Average order value was **₹268.51**, indicating healthy customer spending.

### Monthly Trends

* Highest monthly sales were recorded in **January (₹68.25 Lakhs)**.
* February showed the lowest sales among the months analyzed.
* Sales remained relatively stable after March.

### Weekly Trends

* Saturday generated the highest sales.
* Tuesday recorded the lowest sales.

### Food Preferences

* Veg food contributed **64%** of total sales.
* Non-Veg contributed **36%**.

### Restaurant Performance

* KFC generated the highest revenue.
* McDonald's and Pizza Hut ranked second and third respectively.

### Regional Analysis

* Karnataka generated the highest sales.
* Bengaluru was the highest revenue-generating city.

### Customer Satisfaction

* Average customer rating remained **4.34**, indicating consistently positive customer feedback.
